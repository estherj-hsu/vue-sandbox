## 紀錄

### 2017/07/04

02 主環境初始化
webpack + vue

``` bash
# install vue-cli global
npm install -g vue-cli
# vue + webpack
vue init webpack [project]
# install npm packages
npm i
```
Extra: pug + scss

``` bash
# pug loader & filter
npm install pug pug-loader pug-filters -D
# jage loader
npm install jade jade-loader -D
# sass loader
npm install sass sass-loader node-sass -D
```
